# Sistema de Gestão Académica - Protótipo Frontend

Este repositório contém o código-fonte do protótipo frontend para o Sistema de Gestão Académica, desenvolvido em React e estilizado com Tailwind CSS.

## Visão Geral

Este protótipo simula as principais funcionalidades de um sistema de gestão académica, incluindo:
-   **Dashboard** com indicadores visuais.
-   **Gestão de Cursos** (CRUD completo).
-   **Gestão de Alunos** (CRUD completo).
-   **Gestão de Eventos e Planeamento de Mídia**, permitindo associar ações de comunicação a cada evento.

## Tecnologias Utilizadas

-   [React](https://reactjs.org/)
-   [Tailwind CSS](https://tailwindcss.com/)
-   [Recharts](https://recharts.org/) para gráficos.
-   [Lucide React](https://lucide.dev/) para ícones.

## Como Executar o Projeto

### Pré-requisitos

-   [Node.js](https://nodejs.org/) (versão 14 ou superior)
-   [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

### Instalação

1.  Clone o repositório:
    ```sh
    git clone <URL_DO_SEU_REPOSITORIO_GIT>
    ```
2.  Navegue para a pasta do projeto:
    ```sh
    cd <NOME_DA_PASTA_DO_PROJETO>
    ```
3.  Instale as dependências:
    ```sh
    npm install
    ```

### Executar a Aplicação

Para iniciar o servidor de desenvolvimento, execute:
```sh
npm start
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver a aplicação em funcionamento.
